# Scanner 模块文件说明

## 模块总览

`src-tauri/src/scanner/` 包含所有与文件扫描、分析相关的 Rust 代码。每个子模块职责清晰、独立可测。

---

## 文件列表

### `mod.rs`
模块入口文件，声明所有子模块并统一导出公共类型。

### `scan_engine.rs`
**垃圾扫描引擎** — 一键清理的核心扫描逻辑。
- `ScanEngine` — 支持并行扫描、取消、进度推送
- 10种垃圾分类的目录遍历与文件统计

### `categories.rs`
**垃圾分类定义** — 10种垃圾类型的枚举和元数据。
- `JunkCategory` — 类别ID、名称、描述、风险等级
- 每种类别对应的扫描路径和扩展名规则

### `file_info.rs`
**扫描结果数据结构** — 扫描过程中使用的公共类型。
- `FileInfo` — 文件基本信息（路径、大小、修改时间）
- `ScanResult` / `CategoryScanResult` / `ScanProgress`

### `big_files.rs`
**大文件扫描模块** — 系统盘 Top-N 大文件收集。
- `LargeFileEntry` — 含风险等级(1-5)和来源标签(20+种)
- 基于最小堆(BinaryHeap)的Top-N收集算法
- 风险计算：`compute_file_risk_level()` / 来源识别：`compute_source_label()`
- 遍历时跳过 `$Recycle.Bin`、`System Volume Information`、`WinSxS`

### `social_scanner.rs`
**社交软件专清扫描器** — 智能路径溯源与文件分类。
- `SocialScanner` — 主扫描器，支持8款社交软件
- 路径检测：注册表溯源 + 默认目录 + 全盘搜索三级回退
- `classify_file()` — 五规则优先级分类引擎
- `RiskLevel` / `FileCategory` — 风险分级类型
- `SocialScanResult` / `SocialFileEntry` / `SocialCategoryStats` — 扫描结果类型

### `hotspot.rs`
**大目录分析（热点扫描）** — AppData 语义识别。
- `HotspotScanner` — 识别占用空间最大的目录
- 风险标记：安全/谨慎/危险
- 支持目录下钻和一键打开

### `leftovers.rs`
**卸载残留扫描** — 置信度评分引擎。
- `LeftoverScanner` — 7项正向信号 + 7项负向信号
- 结构化应用映射、预过滤降噪
- 模拟器残留检测（雷电/蓝叠/夜神/MuMu等）
- 虚拟磁盘文件识别（.vmdk/.vdi/.vhd）

### `registry.rs`
**注册表残留扫描** — HKCR\Applications 文件关联残留。
- 铁证条件过滤（关联exe不存在 + 非系统路径 + 非系统进程）
- .reg 备份文件生成（reg.exe export）
- `registry_scoring.rs` — 路径解析 / 存在性缓存 / 安全过滤

### `context_menu.rs`
**右键菜单清理** — 扫描与清理 Windows 右键菜单失效项。
- 覆盖所有右键场景（文件/文件夹/桌面/磁盘）
- 自动检查命令引用exe是否存在
- 分权限操作（HKCU不需管理员 / HKLM需管理员）

### `programdata.rs`
**ProgramData 扫描器** — C:\ProgramData 目录分析。
- 两层扫描策略（一级全量 + 超过100MB自动下钻）
- 支持自定义阈值

### `programdata_rules.rs`
**ProgramData 规则引擎** — 内置20+条分类规则。
- `RuleEngine` — 从 JSON 文件或内嵌规则加载
- 风险分级：Safe/Caution/Danger
- 支持规则热加载

### `programdata_cleaner.rs`
**ProgramData 安全清理** — 移动到回收站 + 安全校验。
- `ProgramDataCleaner` + `CleanOptions`
- 批量清理 + 干运行模式

### `programdata_snapshot.rs`
**ProgramData 快照系统** — 目录大小快照存储与检索。
- `SnapshotManager` — 创建/加载/对比快照
- 按日期存储，JSON格式

### `programdata_growth.rs`
**ProgramData 增长分析** — 对比快照找出增长最快的目录。
- `GrowthAnalyzer` — 增长分级（Significant/Fast/Moderate/Minor）
- `compare_growth_with_timespan()` — 基于时间跨度的对比

