## вңЁ еҠҹиғҪзү№жҖ§

### рҹ”Қ дёҖй”®жү«жҸҸжё…зҗҶ
- **10з§ҚеһғеңҫеҲҶзұ»**пјҡWindowsдёҙж—¶ж–Үд»¶гҖҒзі»з»ҹзј“еӯҳгҖҒжөҸи§ҲеҷЁзј“еӯҳгҖҒеӣһж”¶з«ҷгҖҒWindowsжӣҙж–°зј“еӯҳгҖҒзј©з•Ҙеӣҫзј“еӯҳгҖҒж—Ҙеҝ—ж–Үд»¶гҖҒеҶ…еӯҳиҪ¬еӮЁгҖҒж—§Windowsе®үиЈ…гҖҒеә”з”Ёзј“еӯҳ
- **еӨҡзәҝзЁӢе№¶иЎҢжү«жҸҸ**пјҡеҲ©з”ЁRustзҡ„й«ҳжҖ§иғҪе№¶еҸ‘иғҪеҠӣпјҢеҝ«йҖҹйҒҚеҺҶж–Үд»¶зі»з»ҹ
- **е®һж—¶иҝӣеәҰеҸҚйҰҲ**пјҡжү«жҸҸиҝҮзЁӢдёӯе®һж—¶жҳҫзӨәеҪ“еүҚеҲҶзұ»е’ҢиҝӣеәҰ
- **жү«жҸҸеҒңжӯўжҺ§еҲ¶**пјҡжү«жҸҸиҝҮзЁӢдёӯеҸҜйҡҸж—¶зӮ№еҮ»еҒңжӯўжҢүй’®з»ҲжӯўжүҖжңүжү«жҸҸд»»еҠЎ
- **иҷҡжӢҹеҲ—иЎЁдјҳеҢ–**пјҡеӨ§йҮҸж–Үд»¶еҲ—иЎЁд№ҹиғҪжөҒз•…ж»ҡеҠЁ

### пҝҪ еӨ§ж–Үд»¶жё…зҗҶ
- **жҷәиғҪжү«жҸҸ**пјҡеҝ«йҖҹиҜҶеҲ« C зӣҳдёӯеҚ з”Ёз©әй—ҙжңҖеӨ§зҡ„ж–Үд»¶
- **йЈҺйҷ©зӯүзә§ж ҮиҜҶ**пјҡж №жҚ®ж–Үд»¶еӨ§е°ҸиҮӘеҠЁж Үи®°йЈҺйҷ©зӯүзә§
- **дёҖй”®е®ҡдҪҚ**пјҡж”ҜжҢҒжү“ејҖж–Үд»¶жүҖеңЁзӣ®еҪ•жҲ–зӣҙжҺҘжү“ејҖж–Үд»¶
- **жү№йҮҸйҖүжӢ©еҲ йҷӨ**пјҡеӢҫйҖүеҗҺдёҖй”®жё…зҗҶпјҢйҮҠж”ҫеӨ§йҮҸз©әй—ҙ

### рҹ’¬ зӨҫдәӨиҪҜд»¶дё“жё…
- **еӨҡе№іеҸ°ж”ҜжҢҒ**пјҡеҫ®дҝЎгҖҒQQгҖҒй’үй’үгҖҒйЈһд№ҰгҖҒдјҒдёҡеҫ®дҝЎзӯүдё»жөҒзӨҫдәӨиҪҜд»¶
- **жҷәиғҪи·Ҝеҫ„жЈҖжөӢ**пјҡиҮӘеҠЁиҜҶеҲ«еҗ„иҪҜд»¶зҡ„зј“еӯҳзӣ®еҪ•пјҲж”ҜжҢҒиҮӘе®ҡд№үе®үиЈ…и·Ҝеҫ„пјү
- **еҲҶзұ»з®ЎзҗҶ**пјҡеӣҫзүҮи§Ҷйў‘гҖҒж–Үд»¶зј“еӯҳгҖҒе…¶д»–зј“еӯҳеҲҶзұ»еұ•зӨә
- **е®үе…Ёжё…зҗҶ**пјҡд»…жё…зҗҶзј“еӯҳж–Үд»¶пјҢдёҚеҪұе“ҚиҒҠеӨ©и®°еҪ•

### рҹҡҖ зі»з»ҹзҳҰиә«пјҲйңҖз®ЎзҗҶе‘ҳжқғйҷҗпјү
- **дј‘зң ж–Үд»¶з®ЎзҗҶ**пјҡдёҖй”®е…ій—ӯ/ејҖеҗҜдј‘зң еҠҹиғҪпјҢйҮҠж”ҫдёҺеҶ…еӯҳзӯүйҮҸзҡ„з©әй—ҙпјҲ8-32GBпјү
- **зі»з»ҹз»„д»¶жё…зҗҶ**пјҡи°ғз”Ё DISM жё…зҗҶ WinSxS з»„д»¶еӯҳеӮЁдёӯзҡ„еҶ—дҪҷж–Үд»¶
- **иҷҡжӢҹеҶ…еӯҳдјҳеҢ–**пјҡжЈҖжөӢеҲҶйЎөж–Үд»¶дҪҚзҪ®пјҢеј•еҜјиҝҒз§»еҲ°йқһзі»з»ҹзӣҳ
- **йЈҺйҷ©жҸҗзӨә**пјҡжҜҸйЎ№ж“ҚдҪңйғҪжңүиҜҰз»Ҷзҡ„еҠҹиғҪиҜҙжҳҺе’ҢйЈҺйҷ©иӯҰе‘Ҡ

### рҹ”¬ еӨ§зӣ®еҪ•еҲҶжһҗ
- **иҜӯд№үиҜҶеҲ«жҠҖжңҜ**пјҡж·ұеәҰеҲҶжһҗ AppData зӣ®еҪ•пјҢжҷәиғҪиҜҶеҲ«еҚ з”Ёз©әй—ҙжңҖеӨ§зҡ„ж–Үд»¶еӨ№
- **е…ғеҮ¶е®ҡдҪҚ**пјҡеҝ«йҖҹжүҫеҮә C зӣҳз©әй—ҙзҡ„"зҪӘйӯҒзҘёйҰ–"пјҢзІҫеҮҶйҮҠж”ҫзЈҒзӣҳз©әй—ҙ
- **йЈҺйҷ©ж Үи®°**пјҡиҮӘеҠЁж Үи®°зЁӢеәҸзј“еӯҳе’ҢжҪңеңЁйЈҺйҷ©йЎ№пјҢиҫ…еҠ©еҶізӯ–
- **дёҖй”®жү“ејҖ**пјҡж”ҜжҢҒзӣҙжҺҘжү“ејҖж–Үд»¶еӨ№жҲ–еңЁзәҝжҗңзҙўдәҶи§ЈжӣҙеӨҡдҝЎжҒҜ

### рҹҺҜ ж·ұеәҰеҚёиҪҪж®Ӣз•ҷжё…зҗҶпјҲзҪ®дҝЎеәҰиҜ„еҲҶеј•ж“Һпјү
- **зҪ®дҝЎеәҰиҜ„еҲҶжЁЎеһӢ**пјҡеҹәзәҝ 0.0 зәҜжӯЈеҗ‘й©ұеҠЁпјҢ7 йЎ№жӯЈеҗ‘дҝЎеҸ·пјҲDisplayName еҢ№й…Қ +0.45гҖҒеҚёиҪҪзЁӢеәҸж®Ӣз•ҷ +0.35гҖҒеҸҜжү§иЎҢж–Үд»¶ +0.20 зӯүпјүе’Ң 7 йЎ№иҙҹеҗ‘дҝЎеҸ·пјҲе·Іе®үиЈ…еә”з”Ё -0.60гҖҒйҖҡз”Ёзӣ®еҪ• -0.40гҖҒProgramData -0.30 зӯүпјүзӢ¬з«ӢеҠ жқғпјҢscore вүҘ 0.65 дёәй«ҳзҪ®дҝЎеәҰж®Ӣз•ҷгҖҒ0.40~0.65 дёәеҸҜз–‘йЎ№
- **з»“жһ„еҢ–еә”з”Ёжҳ е°„**пјҡд»ҺжіЁеҶҢиЎЁ Uninstall й”®жҸҗеҸ– InstallLocation жң«зә§/еҖ’ж•°з¬¬дәҢзә§зӣ®еҪ•еҗҚжһ„е»әзІҫзЎ®жҳ е°„пјҢдёҚеҶҚжӢҶеҲҶ DisplayName tokenпјҢжқңз»қзҹӯиҜҚзў°ж’һиҜҜеҲӨ
- **йў„иҝҮж»ӨйҷҚеҷӘ**пјҡеҢ…еҗҚж јејҸзӣ®еҪ•пјҲ`com.xxx.yyy`пјүе’ҢзәҜзүҲжң¬еҸ·зӣ®еҪ•пјҲ`1.2.3.4`гҖҒ`v2.0`пјүеңЁиҜ„еҲҶеүҚзӣҙжҺҘи·іиҝҮ
- **з»“жһ„еҢ–зҷҪеҗҚеҚ•**пјҡ`Exact` / `Prefix` / `Pattern` дёүз§Қи§„еҲҷпјҢзҰҒжӯўе…ЁеұҖ `contains` еҢ№й…Қ
- **жЁЎжӢҹеҷЁж®Ӣз•ҷжЈҖжөӢ**пјҡж”ҜжҢҒдё»жөҒе®үеҚ“жЁЎжӢҹеҷЁпјҲ**йӣ·з”өгҖҒи“қеҸ гҖҒеӨңзҘһгҖҒMuMuгҖҒMEmuгҖҒи…ҫи®ҜжүӢжёёеҠ©жүӢ**зӯүпјүпјҢе‘ҪдёӯеҗҺзӣҙжҺҘзҪ®дҝЎеәҰ 0.90
- **иҷҡжӢҹзЈҒзӣҳжү«жҸҸ**пјҡиҮӘеҠЁиҜҶеҲ« `.vmdk`гҖҒ`.vdi`гҖҒ`.vhd` зӯүиҷҡжӢҹзЈҒзӣҳж–Үд»¶пјҢиҝҷдәӣж–Үд»¶йҖҡеёёеҚ з”Ёж•°еҚҒGBз©әй—ҙ
- **жіЁеҶҢиЎЁж·ұеәҰжү«жҸҸ**пјҡжү«жҸҸ HKCU/HKLM Software дёӢзҡ„еӯӨз«ӢжіЁеҶҢиЎЁйЎ№е’ҢеӯӨз«Ӣй©ұеҠЁжңҚеҠЎйЎ№
- **еӨ§ж–Үд»¶й«ҳдә®**пјҡжЁЎжӢҹеҷЁж®Ӣз•ҷе’ҢеӨ§еһӢж–Үд»¶д»ҘзәўиүІй«ҳдә®жҳҫзӨәпјҢж–№дҫҝеҝ«йҖҹиҜҶеҲ«

### пёҸпёҸ еҸій”®иҸңеҚ•жё…зҗҶ
- **ж·ұеәҰжү«жҸҸжіЁеҶҢиЎЁ**пјҡеҹәдәҺ Rust й«ҳжҖ§иғҪ winreg жү«жҸҸеҷЁпјҢиҰҶзӣ–д»»ж„Ҹж–Үд»¶гҖҒж–Үд»¶еӨ№гҖҒжЎҢйқўиғҢжҷҜгҖҒзЈҒзӣҳеӨҮеұұзӯүжүҖжңүеңәжҷҜ
- **жҷәиғҪиҜҶеҲ«еӨұж•ҲйЎ№**пјҡиҮӘеҠЁжЈҖжҹҘиҸңеҚ•е‘Ҫд»Өдёӯеј•з”Ёзҡ„ exe ж–Үд»¶жҳҜеҗҰеӯҳеңЁпјҢй»ҳи®ӨеӢҫйҖүеӨұж•ҲжқЎзӣ®
- **еҲҶз»„еұ•зӨә**пјҡжҢүдҪңз”ЁиҢғеӣҙеұ•зӨәпјҢж”ҜжҢҒеұ•ејҖжқЎзӣ®жҹҘзңӢе®Ңж•ҙжіЁеҶҢиЎЁи·Ҝеҫ„е’ҢеҺҹе§Ӣе‘Ҫд»Ө
- **еҲҶжқғйҷҗж“ҚдҪң**пјҡз”ЁжҲ·зә§пјҲHKCUпјүдёҚйңҖз®ЎзҗҶе‘ҳеҚіеҸҜеҲ йҷӨпјӣзі»з»ҹзә§пјҲHKLMпјүж ҮиҜҶйңҖиҰҒз®ЎзҗҶе‘ҳжқғйҷҗ

### рҹ“Ӯ ProgramData еҲҶжһҗ
- **дёӨеұӮжү«жҸҸзӯ–з•Ҙ**пјҡдёҖзә§зӣ®еҪ•е…ЁйҮҸжү«жҸҸпјҢи¶…иҝҮ 100MB зҡ„зӣ®еҪ•иҮӘеҠЁдёӢй’»еӯҗзӣ®еҪ•
- **и§„еҲҷеј•ж“ҺеҲҶжһҗ**пјҡеҶ…зҪ® 20+ жқЎеҲҶзұ»и§„еҲҷпјҢиҮӘеҠЁиҜҶеҲ« Windows UpdateгҖҒDefenderгҖҒй©ұеҠЁзј“еӯҳгҖҒDockerгҖҒAdobe зӯүзӣ®еҪ•
- **йЈҺйҷ©еҲҶзә§**пјҡе®үе…Ё/и°Ёж…Һ/еҚұйҷ©дёүзә§ж ҮиҜҶпјҢе®үе…ЁйЎ№еҸҜдёҖй”®жё…зҗҶпјҢеҚұйҷ©йЎ№иҮӘеҠЁдҝқжҠӨ
- **еўһй•ҝеҜ№жҜ”**пјҡеҹәдәҺеҝ«з…§зі»з»ҹиҝҪиёӘзӣ®еҪ•еӨ§е°ҸеҸҳеҢ–пјҢжүҫеҮәвҖңжӮ„жӮ„еҸҳеӨ§вҖқзҡ„зӣ®еҪ•
- **е®үе…Ёжё…зҗҶ**пјҡжүҖжңүеҲ йҷӨз§»еҠЁеҲ°еӣһж”¶з«ҷпјҢеӨҡеұӮе®үе…Ёж ЎйӘҢпјҢзЁіе®ҡжҖ§дјҳе…Ҳ

### пҝҪрҹӣЎпёҸ е®үе…ЁдҝқжҠӨ
- **зі»з»ҹи·Ҝеҫ„дҝқжҠӨ**пјҡиҮӘеҠЁиҜҶеҲ«е№¶и·іиҝҮе…ій”®зі»з»ҹж–Үд»¶е’Ңзӣ®еҪ•
- **еӨҡеұӮе®үе…ЁйӘҢиҜҒ**пјҡеҲ йҷӨеүҚиҝӣиЎҢи·Ҝеҫ„еҗҲжі•жҖ§гҖҒжқғйҷҗгҖҒиҢғеӣҙзӯүеӨҡйҮҚж ЎйӘҢ
- **йЈҺйҷ©зӯүзә§ж ҮиҜҶ**пјҡжҜҸдёӘеҲҶзұ»йғҪжңүжҳҺзЎ®зҡ„йЈҺйҷ©зӯүзә§жҸҗзӨәпјҲе®үе…Ё/дҪҺйЈҺйҷ©/дёӯзӯү/й«ҳйЈҺйҷ©пјү
- **ж“ҚдҪңзЎ®и®Ө**пјҡеҚұйҷ©ж“ҚдҪңеүҚеј№еҮәзЎ®и®ӨеҜ№иҜқжЎҶпјҢйҳІжӯўиҜҜеҲ 

### рҹҺЁ зҺ°д»ЈеҢ–з•Ңйқў

| **дј‘зң ж–Үд»¶** | 8-32GBпјҲдёҺеҶ…еӯҳзӯүйҮҸпјү | е…ій—ӯдј‘зң е°ҶеҜјиҮҙеҝ«йҖҹеҗҜеҠЁеҠҹиғҪеӨұж•ҲпјҢз”өи„‘ж— жі•иҝӣе…Ҙдј‘зң зҠ¶жҖҒ |
| **зі»з»ҹз»„д»¶еӯҳеӮЁ** | 1-5GB | жё…зҗҶ WinSxS дёӯзҡ„ж—§зүҲжң¬з»„д»¶пјҢжё…зҗҶеҗҺж— жі•еҚёиҪҪе·Іе®үиЈ…зҡ„жӣҙж–° |

### дҪҝз”Ёж–№жі•

1. **еҸій”®зӮ№еҮ»** LightC зЁӢеәҸеӣҫж Ү
2. йҖүжӢ© **"д»Ҙз®ЎзҗҶе‘ҳиә«д»ҪиҝҗиЎҢ"**
3. иҝӣе…Ҙ **зі»з»ҹзҳҰиә«** йЎөйқў
4. ж №жҚ®йңҖиҰҒзӮ№еҮ»еҗ„йЎ№зҡ„ж“ҚдҪңжҢүй’®

### жҠҖжңҜе®һзҺ°

- **дј‘зң ж–Үд»¶**пјҡи°ғз”Ё `powercfg -h off/on` е‘Ҫд»Ө
- **зі»з»ҹз»„д»¶еӯҳеӮЁ**пјҡи°ғз”Ё `dism.exe /online /cleanup-image /startcomponentcleanup /resetbase`
- **иҷҡжӢҹеҶ…еӯҳ**пјҡиҜ»еҸ–жіЁеҶҢиЎЁжЈҖжөӢеҲҶйЎөж–Үд»¶дҪҚзҪ®пјҢжү“ејҖзі»з»ҹеұһжҖ§й«ҳзә§и®ҫзҪ®
