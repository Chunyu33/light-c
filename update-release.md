# LightC v2.2.0 Release Notes

> 轻量级 Windows C盘智能清理工具

## 🆕 新增功能

### 右键菜单清理（全新）
深度扫描 Windows 注册表中注册的右键菜单项，找出并移除那些指向已不存在可执行文件的失效条目，让右键菜单保持干净整洁。

- **全场景覆盖**：扫描"任意文件""文件夹""桌面背景""磁盘驱动器""库文件夹"等所有右键场景
- **智能失效识别**：基于 Rust 高性能 `winreg` 扫描器，自动检测命令引用的 exe 是否存在于磁盘
- **默认精准勾选**：扫描完成后仅自动选中无效条目，有效条目不受影响
- **分组清晰展示**：按作用范围（文件/文件夹/桌面背景等）分组，支持展开查看完整注册表路径、原始命令字符串
- **分权限操作**：
  - 用户级条目（`HKCU`）——无需管理员，直接删除
  - 系统级条目（`HKLM`）——标注"需管理员"，请以管理员身份运行后操作

## 🎨 体验优化

### 模态框动画升级
设置弹窗及所有二次确认对话框的弹出与关闭动画全面升级：

- **弹入**：采用 `cubic-bezier(0.16, 1, 0.3, 1)` 弹簧曲线 + 轻微位移，280ms 快速收敛，参考 Windows 11 / 微信桌面版窗口弹出质感
- **弹出**：采用 `cubic-bezier(0.4, 0, 1, 1)` 快速收缩曲线，185ms 干脆消失，避免拖拉感
- **遮罩**：淡入淡出独立控制，不再与弹窗内容耦合
- **无额外依赖**：纯 CSS `@keyframes` 实现，零运行时开销

## ⚠️ 注意事项

- 删除右键菜单条目后，对应项目将不再出现在右键菜单中；如需恢复，请手动修改注册表或重新安装对应软件
- 删除 HKLM 条目需要管理员权限，权限不足时会在详情中记录失败原因
- 深度清理功能会直接从磁盘永久删除文件，不经过回收站
- 建议在清理前备份重要数据
- 系统瘦身功能需要以管理员身份运行
