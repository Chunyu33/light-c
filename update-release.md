# LightC v2.2.1 Release Notes

> 轻量级 Windows C盘智能清理工具

📢 版本背景
近期发现 LightC 被多家第三方软件下载平台收集。这些平台往往会通过捆绑插件、修改主页甚至植入后门来获利，可能会损害用户的系统安全和我的软件口碑。

由于本人精力有限，暂无暇逐一处理这些平台。为了保护用户不被误导，本版本（v2.2.1）重点强化了**“官方正版验证”**体系，旨在让所有用户都能一眼识别并下载到纯净、安全的官方原版。

## 🆕 新增功能

### 启动动画（全新）
新增官方正版验证视觉，启动时展示高质感动画，强化正版意识：

- **像素风 Logo**：居中展示像素艺术风格图标，带清透绿色发光效果
- **新海诚风格扫描光束**：从上到下循环的浅绿色光束，带多层 glow 阴影
- **文字淡入动画**：依次显示"Checking File Integrity..."和"Verified by SHA-256 Checksum"
- **品牌背书**：底部常驻"Evan的像素空间 · 官方正版"及防篡改警示
- **双窗口架构**：Tauri 2.0 splashscreen + main 窗口分离，启动体验更流畅

### 安全与校验选项卡
设置中新增独立的"安全与校验"选项卡，帮助用户验证软件来源：

- **官方版本安全声明**：警示第三方打包风险（捆绑插件、主页劫持、后门程序）
- **一键校验工具**：自动生成当前版本的 PowerShell/CMD 校验命令，点击即复制
- **官方渠道标识**：明确标注 GitHub Releases 和 B站 [@Evan的像素空间](https://space.bilibili.com/387797235) 为官方唯一发布与动态更新渠道

### 版权与渠道声明
首页底部新增版权信息及官方渠道提示：

- **版权信息**：© 2026 LightC · Evan的像素空间 · 保留所有权利
- **正版渠道提示**：链接至 GitHub 和 B站官方主页
- **防篡改警示**：提醒用户勿从第三方下载站获取

## 🎨 体验优化

### 健康评分指标提示
鼠标悬停在首页健康评分的三个指标上时，显示详细评分规则：

- **磁盘空间**（满分40）：可用空间 ≥30% → 40分，20-30% → 30分，10-20% → 20分，<10% → 10分
- **休眠文件**（满分30）：已关闭 → 30分，存在 → 0分
- **垃圾文件**（满分30）：<500MB → 30分，500MB-2GB → 20分，2-5GB → 10分，>5GB → 0分

## ⚠️ 注意事项

- 请务必从官方渠道下载，第三方下载站的版本可能已被篡改
- 安装前建议使用校验工具验证文件 SHA-256 值
- 深度清理功能会直接从磁盘永久删除文件，不经过回收站
- 建议在清理前备份重要数据
- 系统瘦身功能需要以管理员身份运行
