# LightC v2.1.0 Release Notes

> 轻量级 Windows C盘智能清理工具

## 🆕 新增功能

### 大目录分析（新增）
- 快速定位 AppData 中占用空间的元凶
- 通过语义识别技术智能分析目录结构
- 自动标记程序缓存和潜在风险项

### 智能下钻分析
- 深度扫描模式下，当目录超过 **5GB** 且包含超过 **1000** 个文件时，自动分析子目录结构
- 最多下钻 **3 层**，展示每层占用空间最大的前 3 个子目录
- 子目录使用缩进和"父目录 > 子目录"路径简写，层级清晰
- 帮助精准定位空间占用来源，快速找到"元凶中的元凶"

### 社交软件专清增强
- **智能路径溯源**：自动读取注册表获取微信/QQ 自定义存储路径
  - 支持 `MyDocument:` 特殊值解析
  - 支持绝对路径（如 `E:\data\xwechat_files`）直接使用
- **全盘搜索备选**：当注册表和默认路径都失败时，自动搜索所有磁盘
- **精准风险分级**：
  - 🔴 **CRITICAL**：聊天记录数据库自动锁定，禁止删除
  - 🟠 **MEDIUM**：传输文件提示谨慎清理
  - 🟢 **LOW**：图片视频缓存建议清理
  - 🔵 **NONE**：临时缓存安全清理
- 支持微信、QQ/NTQQ、钉钉、飞书、企业微信、Telegram

### 深度卸载残留清理
- 支持主流安卓模拟器残留检测（雷电、蓝叠、夜神、MuMu、MEmu、腾讯手游助手）
- 自动识别虚拟磁盘文件（.vmdk/.vdi/.vhd）
- 注册表及驱动级深度残留分析

### 交流与支持
- 新增 QQ 交流群模块，支持一键复制群号
- 赞赏功能升级：打赏码支持点击放大查看

## 🎨 UI 优化

- **桌面应用规范化**：全局禁用文本选中，提升原生体验
- **欢迎弹窗重构**：简约圆角风格，与主界面保持一致
- **术语统一**：将"C盘热点"更名为"大目录分析"
- **下钻结果分级展示**：子目录使用左侧边框缩进，视觉层级清晰
- **社交软件专清卡片简化**：移除冗余的软件标签，界面更简洁

## 🔧 技术改进

- 停用自动更新功能，简化应用架构
- 优化打赏模态框交互，支持 ESC 键关闭
- 打赏码切换带淡入淡出动画效果
- 使用 Rayon 多线程并行扫描，优化大目录分析性能
- 一键扫描采用前端触发器 + 后端 spawn_blocking 并发模式，UI 保持流畅
- 社交软件扫描模块重构为独立的 `social_scanner.rs`，代码结构更清晰

## 📦 安装说明

1. 下载 `LightC_2.1.0_x64-setup.msi` 安装包
2. 双击运行，按提示完成安装
3. 首次运行可能需要管理员权限

## ⚠️ 注意事项

- 深度清理功能会直接从磁盘永久删除文件，不经过回收站
- 建议在清理前备份重要数据
- 系统瘦身功能需要以管理员身份运行
